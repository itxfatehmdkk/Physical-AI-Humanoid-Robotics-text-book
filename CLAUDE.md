# Frontend for Physical AI Textbook

This directory contains the Docusaurus-based frontend for the Physical AI & Humanoid Robotics textbook, including the integrated RAG chatbot.

## Structure
- `docs/` - Markdown files containing textbook content
- `src/` - Custom React components and Docusaurus theme customization
- `static/` - Static assets like images and documents
- `blog/` - Blog posts related to the textbook
- `docusaurus.config.ts` - Configuration for the Docusaurus site
- `sidebars.ts` - Navigation sidebar configuration

## Setup
1. Install dependencies: `npm install` or `yarn install`
2. Run development server: `npm run start` or `yarn start`
3. Build for production: `npm run build` or `yarn build`

## Features
- Interactive textbook content with Docusaurus
- Embedded RAG chatbot powered by the backend API
- Source citations for chatbot responses
- Support for user-selected text queries
- Responsive design for various screen sizes
